FAQs

Frequently Asked Questions

Find answers to common questions about Sidefy features and functionality.

What information sources does Sidefy support?

Sidefy supports a wide range of information sources including system calendars, reminders, RSS feeds, GitHub notifications, and custom event sources. The plugin system allows for virtually unlimited extensibility to add more integrations.

Is my data secure?

Yes, absolutely. Sidefy processes all data locally on your Mac. No information is sent to external servers unless specifically configured by you through plugins that require external data access.

How do I create custom plugins?

We provide comprehensive documentation and a plugin development kit. You can create plugins using JavaScript with system storage, networking, and AI capabilities.

What macOS versions are supported?

Sidefy supports macOS 14.0 (Sonoma) and later versions. The app is optimized for both Intel and Apple Silicon Macs.

Can I customize the appearance?

Yes, Sidefy offers extensive customization options including various themes, color settings, and simplified information display layouts.

How do I get support?

You can reach out through our support email or visit our GitHub repository for community support and plugin development resources.
