Rule Engine: The Future of Event Automation

FeatureRule EngineAutomation
Announcing a powerful rule engine system in Sidefy that will transform how you customize and manage your events.

We’re introducing a groundbreaking Rule Engine system that will revolutionize how you interact with your events. This intelligent automation framework allows you to create custom rules that automatically transform your events based on specific conditions.

Core Capabilities

In the upcoming version, you will get beta access to the Rule Engine. The Rule Engine operates on a simple “if-then” principle: when certain conditions are met, specific actions are automatically applied. In the first version of the Rule Engine, you will be able to create rules with the following capabilities:

  • Match event titles, content, source, or plugin
  • Use exact matches (equals), exact mismatches (notEquals), exact multi-matches (equalsAny), exact multi-mismatches (notEqualsAny), partial matches (contains), partial mismatches (notContains), multi-keyword matches (containsAny), comprehensive exclusions (notContainsAll), all-elements matches (containsAll), exclusion matches (notContainsAny)
  • Combine multiple conditions with AND/OR logic
  • Apply custom colors to matching events
  • Highlight specific text within events
  • Replace text content in events
  • Add custom symbols to events for visual identification
  • Hide unwanted events from your calendar
  • Pin important events to the top

Import & Export

The Rule Engine supports importing and exporting rules through a rules.sidescript file.

This allows you to backup your custom rules, share rule sets with the community, or manage configurations programmatically.

SideScript Syntax (FOR Advanced Users)

For advanced users who want to manually write or edit rules, we have created a comprehensive guide detailing the syntax, operators, and available actions.

Check out the full SideScript Syntax Guide → to learn how to write powerful custom rules.

Migration Note

Once the Rule Engine is implemented, the current keyword highlighting, keyword filtering, and shortcut hide events features will be discontinued. The Rule Engine provides a more flexible and powerful way to implement these functionalities through custom rules.

The Vision

The Rule Engine represents our commitment to making Sidefy truly intelligent. Imagine an app that learns your preferences, automatically organizes events, and presents information exactly how you need it - without any manual effort.

This foundation enables Sidefy to evolve from a simple event viewer into a smart personal assistant that understands your workflow and adapts to your needs.

The architecture supports advanced matching against text patterns, text groups, and length. This enables advanced operations like modifying event colors - transforming how you interact with Sidefy.
