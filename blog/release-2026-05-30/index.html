Redesigned plugin and rules surfaces, a new Magazine detailed bubble layout, RSS category and image detection, GitHub plugin edit sheet, and major performance improvements.

This release refactors core surfaces, adds a Magazine-style detailed bubble layout, extends RSS management, and delivers significant performance improvements.

Performance & Stability

  1. Reduced memory footprint - Significantly lower baseline usage and improved runtime efficiency, bringing typical memory usage down from 1-2 GB to around 200 MB in normal operation
  2. MacPaw OpenAI client - AI translation now uses the MacPaw OpenAI client for more reliable requests and fewer translation failures

UI Refresh

UI Refresh for macOS 26 → Memory Leak Fixes and Stability Improvements Across WebView, Plugins, and RSS →
  1. Redesigned Plugin Manager, Plugin Marketplace, and Rule Editor - Native-aligned hierarchy, materials, and controls
  2. Unified window toolbars, sidebars, and size presets - Consistent chrome and layout options across plugin and rule management windows

Bubbles & Popovers

Clearer Bubble Layout: Magazine Style for Detailed Events →
  1. Magazine detailed layout - Optional reading order with source, title, time, and footer grouped for long entries and plugin events
  2. Color bar gradient toggle - Switch between gradient and flat color bars; flat mode keeps segments proportional to event duration

RSS Feeds

  1. Structured image auto-detection - Feed items with embedded image metadata are picked up automatically for better thumbnail coverage
  2. Category field - Assign categories to feeds and search subscriptions by category
  3. Editable feed descriptions - Update descriptions in Sidefy; changes sync on OPML export

Plugins

  1. GitHub plugin edit sheet - Configure the GitHub plugin in a dedicated edit sheet
