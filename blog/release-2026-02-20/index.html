SideScript 2.0 & Font Customization

Expected Release: Feb 20, 2026
·
Changelog Rule Engine SideScript Font AI Experimental
Introducing SideScript 2.0 with TEMPLATE syntax and WHEN contexts, font customization, AI translation enhancements, and Experimental Features.

This update brings SideScript 2.0 to production and introduces comprehensive font customization options for a personalized reading experience. It also adds AI translation enhancements, theme and appearance options, and a dedicated Experimental Features window.

SideScript 2.0

SideScript 2.0: Extended TEMPLATE Syntax →

  1. TEMPLATE Syntax - Create reusable rule groups that can be referenced across multiple rules
  2. WHEN Contexts - Apply rules based on system conditions like appearance mode and date type
  3. Enhanced rule import/export with version compatibility checking

Font Customization

  1. Adjustable Font Size - Customize event bubble font size (Normal, Large)
  2. Custom Font Import - Import and use your own TTF, OTF, or TTC font files in event bubbles and reader mode

AI Translation

  1. Content translation - Translate event content (not just titles) with AI
  2. Custom translation prompt - Define your own instructions for the AI translator
  3. Show original title - Option to display the original title alongside the translation in the AI translation menu

Theme & Appearance

  1. Random Color Mode - Generate stable random colors for events

Experimental Features

When Experimental Features are enabled (Help menu), a dedicated Experimental Features window is available with:

  1. System logging - Enable release logging and choose log level (Debug / Info / Warning / Error / Critical) for troubleshooting in Console.app
  2. Popover show delay - Adjust the delay before the event popover appears when hovering over the timeline (0–0.5 s). Moved from Advanced Settings in this release.
  3. Code color settings - Customize syntax highlighting colors for SideScript in light and dark themes
  4. Show menu icons (macOS 26+) - Turn on or off icons next to menu items. On macOS 26 and later, the system may add icons automatically; this option lets you turn that off and only show icons you’ve set.

Disabling Experimental Features resets all of these options to their defaults.

Other Updates

  1. Added FAQ menu item in help menu
  2. RSS feed preview functionality
  3. Improved database and font file storage locations
  4. Various bug fixes and performance optimizations
