FAQ

常见问题

了解 Sidefy 的常见问题解答，快速上手并充分利用这款强大的信息聚合工具。

什么是 Sidefy？

Sidefy 是一款 macOS 屏幕边缘信息聚合应用，通过在屏幕边缘常驻显示各种信息源，让您无需切换应用即可随时掌握重要信息。

支持哪些插件？

Sidefy 内置多种插件：系统日历、RSS 订阅、GitHub 活动、提醒事项、使用技巧等。同时支持自定义 JavaScript 插件开发和在线插件商店，满足个性化需求。

如何安装和使用？

从 Mac App Store 下载安装即可。首次启动后，在设置中选择需要的插件，根据个人偏好调整布局和主题。应用会自动在屏幕边缘显示信息。

会影响其他应用的使用吗？

不会。Sidefy 采用智能边缘显示设计，只在屏幕边缘占用少量空间，不会遮挡或干扰其他应用的正常使用。

如何自定义插件？

Sidefy 支持 JavaScript 自定义插件开发。您可以编写自己的插件来显示特定的信息源，或者从在线插件商店下载社区贡献的插件。

系统要求是什么？

Sidefy 需要 macOS 14 或更高版本。应用经过优化，占用系统资源极少，不会影响 Mac 的性能。
