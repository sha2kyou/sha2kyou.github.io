Quick Start

Section titled “Quick Start”

This guide walks you through adding your first RSS feed in Sidefy using the menu bar.

1. Open RSS Plugin Settings from the Menu Bar

Section titled “1. Open RSS Plugin Settings from the Menu Bar”

  1. Click the Sidefy icon in the macOS menu bar (top-right of the screen).

  2. Select Plugins → RSS Reader → Settings.

  3. The RSS Plugin Settings window will open.

Tip

If RSS Reader appears grayed out, enable it first: Plugins → RSS Reader → Enable.

2. Add Your First RSS Feed

Section titled “2. Add Your First RSS Feed”
  1. In the RSS Plugin Settings window, click the + button (top-right).
  2. Choose Add Feed.
  3. Enter an RSS feed URL in the dialog. For example: https://news.ycombinator.com/rss (Hacker News).
  4. Click Add.
  5. Sidefy will fetch and preview the feed. You can optionally edit the title, description, or color.
  6. Click Confirm to add the feed.

3. View RSS Events in the Timeline

Section titled “3. View RSS Events in the Timeline”

After adding a feed, today’s articles from that feed will appear in the Sidefy timeline. Move your cursor to the screen edge to open the sidebar and see today’s RSS news alongside your calendar and reminders.

Note

The URL must be a valid HTTP or HTTPS link. Sidefy supports RSS 2.0, Atom, and JSON Feed formats.
