Frequently Asked Questions

Find answers to common questions about Sidefy features and functionality.

What information sources does Sidefy support?

Sidefy supports calendars, reminders, RSS feeds, GitHub notifications, and custom event sources. The plugin system allows virtually unlimited extensibility.

Is my data secure?

Sidefy processes data locally on your Mac. Nothing is sent to external servers unless you configure a plugin that requires it.

How do I create custom plugins?

See the plugin docs in the footer. Plugins use JavaScript with storage, networking, and AI capabilities.

What macOS versions are supported?

macOS 14.0 (Sonoma) or later, on both Intel and Apple Silicon Macs.

Can I customize the appearance?

Yes — themes, colors, fonts, and layout options are available in Premium.

How do I get support?

Use GitHub or the support email inside the app for help and plugin resources.
