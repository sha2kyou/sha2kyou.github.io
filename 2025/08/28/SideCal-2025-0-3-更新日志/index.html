⏺ SideCal 2025.0.3 更新日志

新功能

  • iCloud 同步：新增跨设备配置同步，自动在多台 Mac 间保持设置一致
  • GitHub 功能整合：统一 Pull Request、Star、Release 等 GitHub 事件显示

安全优化

  • GitHub Personal Access Token 等敏感信息现存储在系统钥匙串中，确保数据安全
  • 智能同步策略：仅同步非敏感配置，保护用户隐私

问题修复

  • 修复分页切换时气泡组件尺寸异常的渲染问题
  • 解决屏幕解锁后数据不刷新的问题
  • 限制设置窗口重复打开，提升界面体验

性能提升

  • 基于时间戳的智能冲突解决，确保多设备数据一致性
  • 优化事件合并算法，减少重复显示
  • 改进界面响应速度和整体稳定性

这个版本主要聚焦于多设备协作体验和数据安全性的提升。
